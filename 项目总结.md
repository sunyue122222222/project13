## 核心文件说明

### C语言程序文件
1. **`fixed_mechanism.c`** - 主要分析程序（推荐使用）
   - 完整的运动学分析
   - 动力学计算
   - 数据输出功能

2. **`six_bar_mechanism.c`** - 基础版本
   - 简化的运动学模型
   - 基本的分析功能

3. **`advanced_mechanism.c`** - 高级版本
   - 更复杂的运动学计算
   - 四杆机构求解算法

### 数据文件
1. **`mechanism_analysis.csv`** - 详细分析数据
   - 360个计算点的完整数据
   - 包含位移、速度、加速度、力学参数

2. **`design_summary.txt`** - 设计说明文档
   - 机构参数总结
   - 分析结果汇总
   - 设计验证结论

### 可视化文件
1. **`mechanism_curves_en.png`** - 英文版运动曲线图
2. **`plot_results_en.py`** - 英文版绘图脚本
3. **`plot_results.py`** - 中文版绘图脚本

## 使用指南

### 快速开始
```bash
# 编译程序
make

# 运行分析
./fixed_mechanism

# 生成图表
python3 plot_results_en.py
```

### 修改参数
在`fixed_mechanism.c`的`init_mechanism_params()`函数中修改：
```c
params->l_OA = 65;      // 机架长度
params->l_AB = 100;     // 连杆长度
params->n2 = 30;        // 转速
params->F_max = 6300;   // 最大工作阻力
```
