# Vivado仿真使用指南

## 🚀 快速开始

### 方法1：使用图形界面（推荐）

1. **添加仿真文件**
   - 在Vivado项目中，右键点击 "Simulation Sources"
   - 选择 "Add Sources" → "Add or create simulation sources"
   - 添加 `vivado_testbench.v` 或 `quick_test.v`

2. **设置顶层模块**
   - 在 "Simulation Sources" 中找到测试文件
   - 右键选择 "Set as Top"

3. **运行仿真**
   - 点击左侧 "SIMULATION" → "Run Simulation" → "Run Behavioral Simulation"
   - 等待仿真启动

4. **观察波形**
   - 在波形窗口中可以看到所有信号
   - 使用工具栏的缩放和导航功能

### 方法2：使用TCL脚本（自动化）

1. **在Vivado TCL Console中运行**：
   ```tcl
   source vivado_sim_setup.tcl
   ```

2. **或者逐步执行**：
   ```tcl
   # 添加仿真文件
   add_files -fileset sim_1 vivado_testbench.v
   
   # 设置顶层
   set_property top vivado_testbench [get_filesets sim_1]
   
   # 启动仿真
   launch_simulation
   
   # 运行仿真
   run all
   ```

## 📊 关键信号监控

### 输入信号
- `clk`: 50MHz系统时钟
- `rst_n`: 复位信号（低有效）
- `key[6:0]`: 按键输入（低有效）

### 输出信号
- `seg[7:0]`: 数码管段选信号
- `dig[5:0]`: 数码管位选信号
- `led[7:0]`: LED指示信号

### 内部状态信号
- `freq_mode[1:0]`: 频率模式（00=1Hz, 01=2Hz, 10=4Hz）
- `precision_mode`: 精度模式（0=1秒, 1=0.1秒）
- `timer_mode`: 计时模式（0=正计时, 1=倒计时）
- `setting_mode`: 设置模式标志
- `count_up[15:0]`: 正计时计数器
- `count_down[15:0]`: 倒计时计数器
- `display_value[15:0]`: 当前显示值

### 时钟信号
- `clk_1hz`: 1Hz计时时钟
- `clk_2hz`: 2Hz计时时钟
- `clk_4hz`: 4Hz计时时钟

## ⚠️ 注意事项

### 仿真设置
1. **时间单位**: 确保使用 `timescale 1ns / 1ps`
2. **仿真时长**: 建议至少50ms以观察完整功能
3. **信号监控**: 添加所有关键信号到波形窗口

### 常见问题
1. **仿真时间过长**: 使用 `quick_test.v` 进行快速验证
2. **信号不显示**: 检查信号路径和层次结构
3. **时钟问题**: 确认时钟频率设置正确

### 性能优化
1. **减少仿真时间**: 调整测试序列中的等待时间
2. **选择性监控**: 只添加需要观察的信号
3. **分段测试**: 可以注释掉部分测试用例
