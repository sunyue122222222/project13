# 六杆机构运动学分析程序

## 项目概述

本项目是一个用C语言编写的六杆机构运动学和动力学分析程序，根据机械设计课程要求，对六杆机构进行全面的运动分析。

## 功能特点

1. **运动学分析**
   - 计算滑块6的位移、速度、加速度
   - 分析机构的运动特性
   - 输出完整的运动曲线数据

2. **动力学分析**
   - 计算工作阻力变化
   - 分析惯性力影响
   - 计算驱动力矩需求

3. **数据可视化**
   - 生成运动曲线图
   - 输出CSV格式数据
   - 提供Python绘图脚本

## 文件结构

```
├── six_bar_mechanism.c      # 基础版本程序
├── fixed_mechanism.c        # 优化版本程序（推荐使用）
├── advanced_mechanism.c     # 高级版本（包含复杂运动学）
├── Makefile                 # 编译配置文件
├── plot_results.py          # Python绘图脚本（中文）
├── plot_results_en.py       # Python绘图脚本（英文）
├── README.md               # 项目说明文档
└── 输出文件/
    ├── mechanism_analysis.csv    # 详细分析数据
    ├── design_summary.txt        # 设计说明文档
    ├── mechanism_curves.png      # 运动曲线图（中文）
    └── mechanism_curves_en.png   # 运动曲线图（英文）
```

## 编译和运行

### 编译程序
```bash
make
# 或者
gcc -Wall -Wextra -std=c99 -O2 -o fixed_mechanism fixed_mechanism.c -lm
```

### 运行分析
```bash
./fixed_mechanism
```

### 生成图表
```bash
# 安装Python依赖
pip install matplotlib pandas

# 生成英文版图表
python3 plot_results_en.py

# 生成中文版图表（可能有字体警告）
python3 plot_results.py
```

## 机构参数

程序使用的机构参数（基于设计表格第1组数据）：

| 参数 | 数值 | 单位 |
|------|------|------|
| 机架长度 l_OA | 65 | mm |
| 连杆长度 l_AB | 100 | mm |
| 连杆长度 l_BC | 16 | mm |
| 方向角 α | 1.6 | 度 |
| 偏心距 e | 1.1 | mm |
| 曲柄转速 n₂ | 30 | r/min |
| 最大工作阻力 | 6300 | N |
| 滑块质量 m₆ | 50 | kg |

## 分析结果

### 运动学特性
- **滑块行程**: 329.97 mm
- **位移范围**: -163.88 ~ 166.08 mm
- **最大速度**: 518.31 mm/s
- **最大加速度**: 1628.33 mm/s²

### 工作特性
- 工作阻力在特定行程段内为6300N
- 空行程时无工作阻力
- 机构运动平稳，满足设计要求
