# FPGA数码管秒表计时器

这是一个基于FPGA的数码管显示秒表计时功能项目，支持正计时、倒计时、频率调整和精度调整等功能。

## 功能特性

### 基本功能
1. **正计时功能**：KEY0控制，从0开始计时，支持暂停/继续
2. **倒计时功能**：KEY1控制，从19秒开始倒计时，支持暂停/继续
3. **模式切换**：KEY2切换正计时和倒计时模式
4. **数码管显示**：6位数码管10进制显示时间

### 扩展功能
1. **初始值设置**：KEY4进入设置模式，可调整倒计时初始值（>10秒）
2. **频率调整**：KEY5循环调整计时频率（1Hz→2Hz→4Hz→1Hz）
3. **精度调整**：KEY6切换显示精度（1秒↔0.1秒）
4. **LED指示**：
   - LED0-1：频率模式指示
   - LED2：精度模式指示
   - LED3：计时模式指示
   - LED4：设置模式指示（闪烁）
   - LED5-7：倒计时警告（最后3秒闪烁）

## 模块说明

### 1. stopwatch_top.v
- **功能**：系统顶层模块，协调各子模块工作
- **输入**：时钟、复位、按键信号
- **输出**：数码管控制信号、LED指示信号

### 2. clock_divider.v
- **功能**：时钟分频，生成1Hz、2Hz、4Hz时钟
- **特点**：支持0.1秒精度的计时

### 3. key_debounce.v
- **功能**：按键消抖和单脉冲生成
- **参数**：20ms消抖时间

### 4. seg_display.v
- **功能**：6位数码管动态扫描显示
- **支持**：秒模式和0.1秒精度模式
- **特点**：前导零消隐、小数点显示

### 5. led_controller.v
- **功能**：LED状态指示和闪烁控制
- **指示**：系统状态、模式切换、警告提示

## 使用说明

### 按键功能
- **KEY0**：正计时开始/暂停，设置模式下增加初始值
- **KEY1**：倒计时开始/暂停，设置模式下减少初始值
- **KEY2**：切换正计时/倒计时模式
- **KEY4**：进入/退出倒计时初始值设置模式
- **KEY5**：循环调整计时频率（1Hz→2Hz→4Hz）
- **KEY6**：切换显示精度（1秒↔0.1秒）
- **复位键**：系统复位，恢复所有初始状态

### 操作流程

#### 正计时模式
1. 确保系统在正计时模式（LED3熄灭）
2. 按KEY0开始计时
3. 再次按KEY0暂停计时
4. 按KEY1复位计时器到0

#### 倒计时模式
1. 按KEY2切换到倒计时模式（LED3点亮）
2. 按KEY1开始倒计时（默认19秒）
3. 再次按KEY1暂停倒计时
4. 按KEY0复位到初始值

#### 设置倒计时初始值
1. 在倒计时模式下按KEY4进入设置模式（LED4闪烁）
2. 按KEY0增加初始值，按KEY1减少初始值
3. 再次按KEY4退出设置模式并开始倒计时

#### 调整频率和精度
- 按KEY5循环调整频率：1Hz→2Hz→4Hz→1Hz
- 按KEY6切换精度：1秒↔0.1秒

## FPGA实现

### 资源需求
- **逻辑单元**：约200个LUT
- **寄存器**：约150个FF
- **时钟域**：1个（50MHz系统时钟）
- **I/O引脚**：21个（7个按键 + 8个数码管段选 + 6个数码管位选）

### 约束文件
项目包含完整的约束文件`constraints.xdc`，适用于Xilinx FPGA开发板。

### 时序要求
- **系统时钟**：50MHz
- **建立时间**：2ns
- **保持时间**：0ns

## 注意事项

1. **按键电平**：按键输入为低有效（按下为0，释放为1）
2. **数码管类型**：支持共阴极数码管
3. **时钟频率**：设计基于50MHz系统时钟
4. **复位方式**：异步复位，低电平有效
5. **显示范围**：
   - 秒模式：0-9999秒
   - 0.1秒模式：0-999.9秒
