# 快速开始指南

## 🚀 快速部署

### 1. 环境准备
```bash
# 安装Verilog仿真工具
sudo apt update
sudo apt install -y iverilog gtkwave

# 克隆项目
git clone <repository_url>
cd project13
```

### 2. 仿真测试
```bash
# 语法检查
make syntax

# 编译并仿真
make all

# 查看波形（可选）
make wave
```

### 3. FPGA实现
1. 打开Vivado或其他FPGA开发工具
2. 创建新项目，添加所有.v文件
3. 导入约束文件 `constraints.xdc`
4. 综合、实现、生成比特流
5. 下载到FPGA开发板

## 🎮 操作指南

### 按键功能速查表

| 按键 | 功能 | 说明 |
|------|------|------|
| KEY0 | 正计时开始/暂停 | 设置模式下：增加初始值 |
| KEY1 | 倒计时开始/暂停 | 设置模式下：减少初始值 |
| KEY2 | 模式切换 | 正计时 ↔ 倒计时 |
| KEY4 | 设置模式 | 进入/退出倒计时初始值设置 |
| KEY5 | 频率调整 | 1Hz → 2Hz → 4Hz → 1Hz |
| KEY6 | 精度调整 | 1秒 ↔ 0.1秒 |
| 复位键 | 系统复位 | 恢复所有初始状态 |

### LED指示速查表

| LED | 功能 | 状态说明 |
|-----|------|----------|
| LED0-1 | 频率模式 | 01=1Hz, 10=2Hz, 11=4Hz |
| LED2 | 精度模式 | 0=1秒, 1=0.1秒 |
| LED3 | 计时模式 | 0=正计时, 1=倒计时 |
| LED4 | 设置模式 | 闪烁=设置中 |
| LED5-7 | 倒计时警告 | 最后3秒闪烁 |

## 📋 使用场景

### 场景1：基本正计时
1. 确保LED3熄灭（正计时模式）
2. 按KEY0开始计时
3. 按KEY0暂停/继续
4. 按KEY1复位到0

### 场景2：基本倒计时
1. 按KEY2切换到倒计时模式（LED3亮）
2. 按KEY1开始倒计时（默认19秒）
3. 按KEY1暂停/继续
4. 按KEY0复位到初始值

### 场景3：自定义倒计时
1. 切换到倒计时模式
2. 按KEY4进入设置模式（LED4闪烁）
3. 用KEY0/KEY1调整初始值（>10秒）
4. 按KEY4退出设置并开始倒计时

### 场景4：高精度计时
1. 按KEY6切换到0.1秒精度（LED2亮）
2. 数码管显示XXX.X格式
3. 按KEY5调整计时频率
4. 正常使用计时功能

## 🔧 故障排除

### 常见问题

**Q: 数码管不显示**
- 检查时钟信号是否正常
- 确认复位信号已释放
- 检查数码管连接

**Q: 按键无响应**
- 确认按键连接正确（低有效）
- 检查消抖时间设置
- 验证按键脉冲生成

**Q: 计时不准确**
- 检查时钟分频参数
- 确认系统时钟频率（50MHz）
- 验证计时逻辑

**Q: LED指示错误**
- 检查LED控制逻辑
- 确认状态变量更新
- 验证闪烁时钟

### 调试技巧

1. **使用仿真**: 先在仿真中验证功能
2. **分模块测试**: 逐个模块验证
3. **信号观察**: 使用示波器或逻辑分析仪
4. **状态监控**: 通过LED观察系统状态

## 📊 性能参数

### 系统规格
- **工作频率**: 50MHz
- **计时精度**: 0.1秒
- **显示范围**: 0-999.9秒（0.1秒模式）或0-9999秒（1秒模式）
- **响应时间**: <20ms（按键消抖）
- **资源占用**: ~200 LUT, ~150 FF

### 功耗估算
- **静态功耗**: <10mW
- **动态功耗**: <50mW
- **总功耗**: <60mW
