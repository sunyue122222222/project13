# FPGA秒表计时器设计规格书

## 1. 项目概述

本项目实现了一个基于FPGA的数码管显示秒表计时器，满足所有基本要求和扩展要求。

### 1.1 设计目标
- 实现正计时和倒计时功能
- 支持多种计时频率和显示精度
- 提供直观的LED状态指示
- 支持用户自定义倒计时初始值

### 1.2 技术规格
- **目标平台**: Xilinx FPGA开发板
- **系统时钟**: 50MHz
- **显示器件**: 6位共阴极数码管
- **输入接口**: 7个按键（KEY0-KEY6）
- **输出接口**: 8个LED指示灯

## 2. 功能需求分析

### 2.1 基本功能需求

#### 2.1.1 正计时功能（需求1）
- **触发条件**: KEY0=1时开始计时，KEY0=0时暂停
- **计时范围**: 从0开始，最大支持9999秒
- **显示格式**: 数码管10进制显示
- **复位方式**: 按复位键恢复到0

#### 2.1.2 倒计时功能（需求2）
- **触发条件**: KEY1=1时开始倒计时，KEY1=0时暂停
- **初始值**: 19秒
- **计时范围**: 从初始值倒数到0
- **复位方式**: 按复位键恢复到初始值

#### 2.1.3 模式切换功能（需求3）
- **触发条件**: KEY2作为控制键
- **切换方式**: 按下KEY2在正计时和倒计时模式间切换
- **状态指示**: LED3指示当前模式

### 2.2 扩展功能需求

#### 2.2.1 初始值设置功能（扩展需求1）
- **进入条件**: KEY4=0进入设置模式
- **设置范围**: 初始值必须大于10秒
- **调整方式**: 
  - KEY0增加初始值
  - KEY1减少初始值
- **退出条件**: KEY4=1退出设置并开始倒计时
- **状态指示**: LED4闪烁表示设置模式

#### 2.2.2 频率调整功能（扩展需求2）
- **触发条件**: 按KEY5循环调整
- **频率序列**: 1Hz → 2Hz → 4Hz → 1Hz
- **状态指示**: LED0-1显示当前频率模式

#### 2.2.3 精度调整功能（扩展需求3）
- **触发条件**: 按KEY6切换
- **精度模式**: 1秒 ↔ 0.1秒
- **显示格式**: 
  - 1秒模式: XXXX（整秒）
  - 0.1秒模式: XXX.X（带小数点）
- **状态指示**: LED2显示当前精度模式

#### 2.2.4 LED警告功能
- **触发条件**: 倒计时最后3秒
- **显示方式**: LED5-7闪烁提示
- **闪烁频率**: 2Hz

## 3. 系统架构设计

### 3.1 模块层次结构

```
stopwatch_top (顶层模块)
├── clock_divider (时钟分频模块)
├── key_debounce × 7 (按键消抖模块)
├── seg_display (数码管显示模块)
└── led_controller (LED控制模块)
```

### 3.2 信号接口定义

#### 3.2.1 顶层接口
```verilog
module stopwatch_top(
    input wire clk,           // 50MHz系统时钟
    input wire rst_n,         // 复位信号（低有效）
    input wire [6:0] key,     // 按键输入
    output wire [7:0] seg,    // 数码管段选
    output wire [5:0] dig,    // 数码管位选
    output wire [7:0] led     // LED输出
);
```

#### 3.2.2 内部信号
- `freq_mode[1:0]`: 频率模式选择
- `precision_mode`: 精度模式选择
- `timer_mode`: 计时模式选择
- `setting_mode`: 设置模式标志
- `count_up[15:0]`: 正计时计数器
- `count_down[15:0]`: 倒计时计数器
- `initial_value[15:0]`: 倒计时初始值

### 3.3 状态机设计

#### 3.3.1 主控制状态机
```
IDLE → COUNT_UP → PAUSE_UP → IDLE
     → COUNT_DOWN → PAUSE_DOWN → IDLE
     → SETTING → IDLE
```

#### 3.3.2 频率切换状态机
```
FREQ_1HZ → FREQ_2HZ → FREQ_4HZ → FREQ_1HZ
```

## 4. 详细设计

### 4.1 时钟分频模块 (clock_divider.v)

#### 4.1.1 功能描述
- 从50MHz系统时钟生成1Hz、2Hz、4Hz时钟
- 实际生成10Hz、20Hz、40Hz以支持0.1秒精度

#### 4.1.2 分频参数
```verilog
parameter CNT_1HZ = 26'd25_000_000;  // 0.1秒分频
parameter CNT_2HZ = 25'd12_500_000;  // 0.05秒分频
parameter CNT_4HZ = 24'd6_250_000;   // 0.025秒分频
```

### 4.2 按键消抖模块 (key_debounce.v)

#### 4.2.1 功能描述
- 消除按键机械抖动
- 生成单脉冲信号和稳定状态信号

#### 4.2.2 消抖参数
```verilog
parameter DEBOUNCE_TIME = 20'd1_000_000;  // 20ms @ 50MHz
```

### 4.3 数码管显示模块 (seg_display.v)

#### 4.3.1 功能描述
- 6位数码管动态扫描显示
- 支持秒模式和0.1秒精度模式
- 前导零消隐和小数点显示

#### 4.3.2 显示格式
- **秒模式**: 显示整秒值（0-9999）
- **0.1秒模式**: 显示XXX.X格式（0.0-999.9）

#### 4.3.3 扫描频率
```verilog
parameter SCAN_FREQ = 16'd1000;  // 1kHz扫描频率
```

### 4.4 LED控制模块 (led_controller.v)

#### 4.4.1 LED功能分配
- **LED0-1**: 频率模式指示
  - 01: 1Hz模式
  - 10: 2Hz模式
  - 11: 4Hz模式
- **LED2**: 精度模式指示（1=0.1秒，0=1秒）
- **LED3**: 计时模式指示（1=倒计时，0=正计时）
- **LED4**: 设置模式指示（闪烁）
- **LED5-7**: 倒计时警告（最后3秒闪烁）

## 5. 时序设计

### 5.1 时钟域
- **主时钟域**: 50MHz系统时钟
- **计时时钟域**: 可选1Hz/2Hz/4Hz时钟
- **显示时钟域**: 1kHz扫描时钟

### 5.3 时序约束

- **时钟偏斜**: <500ps

## 8. 使用说明

### 8.1 操作步骤

#### 8.1.1 正计时操作
1. 确保系统在正计时模式（LED3熄灭）
2. 按KEY0开始计时
3. 再按KEY0暂停计时
4. 按KEY1复位到0

#### 8.1.2 倒计时操作
1. 按KEY2切换到倒计时模式（LED3点亮）
2. 按KEY1开始倒计时
3. 再按KEY1暂停倒计时
4. 按KEY0复位到初始值

#### 8.1.3 设置初始值
1. 在倒计时模式下按KEY4进入设置模式
2. 按KEY0增加，按KEY1减少初始值
3. 再按KEY4退出设置并开始倒计时

#### 8.1.4 调整参数
- 按KEY5循环调整频率
- 按KEY6切换显示精度

### 8.2 状态指示
- **LED0-1**: 频率模式
- **LED2**: 精度模式
- **LED3**: 计时模式
- **LED4**: 设置模式（闪烁）
- **LED5-7**: 倒计时警告
